# DPDP Pre-Audit Website Scanner

A single-page static landing page for early GTM validation of a DPDP pre-audit website review tool designed for auditors, compliance consultants, and privacy advisors.

## Overview

This is a minimal, professional landing page built to test market interest among DPDP auditors and compliance professionals. The page emphasizes professional trust, explicit scope boundaries, and audit workflow alignment.

## Technical Details

- **Single file**: `index.html` (plain HTML + minimal CSS)
- **No dependencies**: No frameworks or external libraries
- **Mobile responsive**: Works on all device sizes
- **System fonts**: Uses native system fonts for fast loading
- **SEO optimized**: Includes meta tags and semantic HTML structure

## Deployment

This page is designed to be hosted on GitHub Pages:

1. Push the repository to GitHub
2. Go to repository Settings → Pages
3. Select the `main` branch as the source
4. The site will be available at `https://[username].github.io/dpdp-preaudit-scanner/`

## Local Development

Simply open `index.html` in a web browser. No build process or server required.
